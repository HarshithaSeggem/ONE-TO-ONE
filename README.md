# ❤️ Heartbeat Classification Using RNN

An Artificial Intelligence based ECG Heartbeat Classification System using a Recurrent Neural Network (RNN) and Streamlit.

The application analyzes ECG signals from the MIT-BIH Arrhythmia Dataset and predicts the heartbeat category with confidence score.

---

## 🚀 Project Overview

Heart diseases are one of the major health challenges worldwide. ECG signals contain important information about heart activity.

This project uses Deep Learning to classify ECG heartbeat signals into different categories automatically.

The system provides:

- ECG signal visualization
- AI-based heartbeat prediction
- Prediction confidence
- Probability distribution of classes
- User-friendly Streamlit interface

---

## 🧠 Model Details

**Deep Learning Model:**

- Simple Recurrent Neural Network (RNN)

**Input:**

- ECG Signal
- 187 signal values

**Output Classes:**

| Class | Heartbeat Type |
|---|---|
| 0 | Normal Beat |
| 1 | Supraventricular Beat |
| 2 | Ventricular Beat |
| 3 | Fusion Beat |
| 4 | Unknown Beat |
